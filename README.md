Developed a Library Book Request application in ServiceNow to automate the process of requesting books from a library. The application allows users to submit book requests through a catalog item, captures details such as book name, author, and request type, and routes the request for approval. After approval, the request is assigned to the library staff for processing. Notifications are sent to users at each stage, and the request status can be tracked until completion. The project was built using ServiceNow features such as Application Studio, Tables, Forms, Service Catalog, Flow Designer, Notifications, Roles, and Reports, improving efficiency and reducing manual work.
